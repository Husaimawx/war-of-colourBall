## ⌨️项目结构

```
├─ assets 代码文件夹
│  ├─ scene 场景文件夹
│  │  ├─ Game 主要游戏场景
│  │  ├─ GameOver 游戏结束场景
│  │  └─ Login 游戏开始场景
│  ├─ scripts Game场景的组件文件夹
│  │  ├─ BonusManager 道具，拾起'Tools'生成'Bonus'技能
│  │  ├─ EnemyManager 敌人
│  │  ├─ Player 玩家
│  │  ├─ Rocker 摇杆
│  │  └─ Score 积分
│  └─ textures 资源文件夹
├─ settings Cocos项目设置文件夹
└─ Cocos杂项文件...
```
